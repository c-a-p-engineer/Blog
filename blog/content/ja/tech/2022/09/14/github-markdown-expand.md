# 【GitHub】GitHub 上の Markdown で表示、非表示を切り替える開閉ボタンを使用する
GitHub 上の Markdown で表示、非表示を切り替える開閉ボタンを使用する

## 開閉ボタン
`details` に囲まれた部分が非表示になります。
`summary` がボタンになります。

```md:expand.md
<details>
<summary>Click!!</summary>
Open!!
</details>
```
